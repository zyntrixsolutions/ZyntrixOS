Official ZyntrixOS update repo

Install once. Update from here.

This repository is the public entrypoint for ZyntrixOS packages. Use it to add the official APT source, track the latest stable build, and keep your system updating from the same place your packages are published.

Quick install

Best for new systems. This downloads the repo installer from this page, adds the signing key, writes the APT source, and refreshes package metadata.

curl -fsSL https://zyntrixsolutions.github.io/ZyntrixOS/setup-repo.sh | sudo bash -s -- --repo-url https://zyntrixsolutions.github.io/ZyntrixOS --channel stable

New install

Start here if this machine has never been pointed at the ZyntrixOS package repo.

sudo apt update && sudo apt install zyntrix-update zyntrix-shell zyntrix-cli zyntrix-config zyntrix-os-base

Update existing system

If the repo is already configured, this pulls the latest published packages from here.

sudo apt update && sudo apt full-upgrade

Switch or reapply channel

Run the installer again with a channel to re-point the machine at this repo cleanly.

curl -fsSL https://zyntrixsolutions.github.io/ZyntrixOS/setup-repo.sh | sudo bash -s -- --repo-url https://zyntrixsolutions.github.io/ZyntrixOS --channel stable

How updates work

Everything a client needs is hosted in this repo.

1

Add the source

`setup-repo.sh` installs `zyntrix-archive-keyring.gpg` and writes the repo entry into `/etc/apt/sources.list.d/zyntrix.list`.

2

Refresh package metadata

A normal `apt update` fetches the signed metadata from `dists/stable/` and sees any newly published versions.

3

Upgrade from the same source

`apt full-upgrade` installs the latest packages from this repo, so users keep updating from the same public entrypoint they installed from.

Manual APT setup

If you prefer to inspect each step yourself.

curl -fsSL https://zyntrixsolutions.github.io/ZyntrixOS/zyntrix-archive-keyring.gpg | sudo tee /usr/share/keyrings/zyntrix-archive-keyring.gpg >/dev/null
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/zyntrix-archive-keyring.gpg] https://zyntrixsolutions.github.io/ZyntrixOS stable main" | sudo tee /etc/apt/sources.list.d/zyntrix.list
sudo apt update && sudo apt full-upgrade
