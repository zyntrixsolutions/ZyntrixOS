ZyntrixOS Package Repository

Static APT repository for ZyntrixOS system updates.

Repository Info

  • Channel: stable
  • Version: 0.1.0-1
  • Architecture: amd64
  • URL: https://zyntrixsolutions.github.io/zyntrixOS

Available Packages

  • zyntrix-update — Update daemon and CLI
  • zyntrix-shell — Desktop UI and shell components
  • zyntrix-cli — Developer CLI and templates
  • zyntrix-config — WM configs and theming
  • zyntrix-os-base — OS branding assets

Quick Setup

On a ZyntrixOS system, run:

sudo ./scripts/setup-repo.sh \
  --repo-url https://zyntrixsolutions.github.io/zyntrixOS \
  --channel stable

Manual APT Setup

Add to /etc/apt/sources.list.d/zyntrix.list:

deb [arch=amd64] https://zyntrixsolutions.github.io/zyntrixOS stable main
